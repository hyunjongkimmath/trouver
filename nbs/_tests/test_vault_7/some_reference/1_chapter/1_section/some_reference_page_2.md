# Topic[^1]
A group $G$ has a unique identity element.

# Proof
Suppose that $1$ and $1'$ are both identity elements of a group $G$. Then,

$$1 \cdot 1' = 1$$

since $1'$ is an identity element. On the other hand,

$$1' \cdot 1 = 1'$$

since $1$ is an identity element. Therefore, $1 = 1'$.
