# Topic[^1]
\begin{proposition}  \label{arithmeticimageproposition}  If  the image  of
$\rho $  is an arithmetic subgroup  of $GL(V)$, then the  image of the
composite $\theta \circ \rho $ is an arithmetic subgroup of $GL(V')$.
\end{proposition}
