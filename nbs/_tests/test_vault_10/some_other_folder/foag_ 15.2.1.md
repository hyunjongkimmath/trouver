15.2.1. Definition. Motivated by this, if $S_{\bullet}$ is a graded ring generated in degree 1 , we define **$\mathscr{O}_{\text {Proj } S_{*}}(n)$** (or simply **$\mathscr{O}(n)$**, where $S_{\bullet}$ is implicit) by $\widehat{S(n)}$
